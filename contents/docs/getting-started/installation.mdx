To get started with this theme, follow these simple steps:

## Prerequisites
Before installing the theme, make sure you have the following:

- Discord desktop app with [BetterDiscord](https://betterdiscord.app/) or [Vencord](https://vencord.dev) injected
- [MicaForEveryone](https://github.com/MicaForEveryone/MicaForEveryone/releases) installed

## Installation Steps

1. **Download the theme from GitHub**: Download our theme from GitHub

   ```bash
   https://github.com/WarFiN123/TranslucenceV2/releases/tag/v
   ```

2. **Navigate to the the themes directory**: Change directory to your themes folder.

   1. Open Discord
   2. Go to settings
   3. Scroll down and find 'Themes'
   4. Click on 'Open Themes Folder'

3. **Add the Theme to Discord**: Paste the downloaded files in the Themes folder and enable it

4. **Install MicaForEveryone**: Download and install MicaForEveryone
   <Note title="Important" type="danger">
   Installing MicaForEveryone is crucial for the Acrylic effect
   </Note>

   - Go to [MicaForEveryone](https://github.com/MicaForEveryone/MicaForEveryone/releases) and get the latest release

5. **Configure MicaForEveryone**: Change settings for MicaForEveryone

    Please copy down the **EXACT** same settings as given below to avoid any errors
   </Note>
